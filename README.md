# A Node-Observability-Aware Dual Fingerprint Greedy Matching Pursuit Method for Hierarchical Location of Harmonic Sources

This repository contains the source code and supporting materials for the study:

**A Node-Observability-Aware Dual Fingerprint Greedy Matching Pursuit Method for Hierarchical Location of Harmonic Sources**

## Overview

This project develops a hierarchical harmonic source location framework under low-observability conditions. The method combines node observability analysis, physical fingerprints, proportional fingerprints, and greedy matching pursuit to progressively identify harmonic source regions and refine source-node estimation.

## Main Features

- Node-observability-aware candidate-space reduction
- Dual-fingerprint construction:
  - Physical fingerprint
  - Proportional fingerprint
- Hierarchical localization for boundary and internal nodes
- Greedy sparse matching in fingerprint space
- Simulation-based validation under limited measurement conditions
